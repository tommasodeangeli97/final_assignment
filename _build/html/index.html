Move_base_client Module

ROS node to interact with the move_base action, it’s composed by two functions, one is to control the modality of the application and the other is to interact with the move_base action

Actions:

move_base to make the robot go to the indicated point

Obstacles Module

ROS node to avoid the collision with the obstacles, continously running

Subscribes to:

/scan where the robot see the laser scanner

Subscribes to:

/cmd_vel where is written the velocity of the robot

Publishes to:

/cmd_vel to change the velocity of the robot
